# 3 Prime Factorization

Main code is below.

```js
let num = 77;
let pri = [2, 3, 5, 7, 11, 13, 17];
let output = [];

let i = 0;

while (i < pri.length) {
  while (num % pri[i] === 0) {
    num = num / pri[i];
    output.push(pri[i]);
  }
  i++;
}

console.log(output);
```
